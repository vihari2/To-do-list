# To do List App

Aplicação de lista de tarefas desenvolvida como projeto de portfólio.

## Tecnologias

**Frontend**

- Vue 3 + Vite
- CSS puro
- Deploy no GitHub Pages

**Backend**

- Node.js + Express
- Deploy no Render

## Funcionalidades

- Criar, listar e deletar tarefas
- Integração com API REST

## Como rodar localmente

**Frontend**

```bash
cd frontend
npm install
npm run dev
```

**Backend**

```bash
cd backend
npm install
node server.js
```
